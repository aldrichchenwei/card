# peasant
一套微信电子名片系统。2013年给科众公司开发的，当时微信公众平台出来不久，第三方微信开发公司如雨后春笋，写这么个简单系统还很多人用。

模板我是直接扒的当时小米商城的，这个系统是一天加一个通宵时间整出来的。
